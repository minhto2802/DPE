Summary

Subpopulation shifts‚Äîdiscrepancies in the distribution of subgroups between training and test data‚Äîcan severely degrade model performance. We propose Diversified Prototypical Ensembles (DPE), a method that replaces the linear classifier with a set of prototype-based classifiers trained to capture diverse latent structures.

DPE is trained in two stages: first using ERM to learn representations, then fine-tuning diversified prototype heads using bootstrapped data subsets and a cosine-based diversity loss (IPS). Our approach improves worst-group accuracy (WGA) across 9 challenging datasets‚Äîwithout needing subgroup labels.
